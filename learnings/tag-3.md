### Was ich heute gelernt habe

**Kern-Konzept: SKILLS**
- Skills = wiederverwendbare Anweisungen in Markdown-Format
- "Skills haben Prozess-Logik, Prompts haben Anweisungen" (mein Zitat!)
- Skill-Struktur: Zweck, Input, Prozess, Qualitätskriterien, Stilregeln, Output-Format

**Wie schreibt man gute Prozesse:**
- "Was passiert in meinem Kopf?" Methode
- "WAS, WIE, WARUM" für jeden Schritt
- NICHT/SONDERN Beispiele sind Gold!

**Modell-Wahl (Haiku vs. Sonnet):**
- Haiku = besser für klar definierte, repetitive Tasks
- Haiku folgt Skills präziser (Instruction Following Champion)
- Sonnet = besser für kreative, komplexe Probleme
- **Meine Entdeckung:** Das günstigere Modell war präziser!

**Iteration ist normal:**
- Skill schreiben → Testen → Fehler finden → Verbessern → Wiederholen
